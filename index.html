Memory game

This is a project with practice in Javascript knowledge.

The rules are simple: Flip over cards by click. At the same time it is possible to open just two cards. If those cards have the same pics - they will remain opened, instead they will close. Player has to open the whole map.

The speed of success depends on the strength of memory. As the player's pace slows down - stars on top fall down.

You can play it here https://kubr2017.github.io/memory-game/


The game is implemented with Javascript code (Array,Map).

Code is here:https://github.com/kubr2017/memory-game

Neighborhood map

This is a project with practice in using React framework.
